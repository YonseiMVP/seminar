Colorization

2017/08/18

발표자: 황상원

[발표 자료]

흑백 영상에 색을 입히려는 시도는 과거에도 많았습니다. 하지만 CNN 의 혁명적인 발전과 더불어 ImageNet과 같은 방대한 Database 덕분에 현재 Colorization 연구는 상당히 진전된 상황입니다. 이번 세미나에서는 Colorization의 문제를 과거서 부터 현재까지 어떠한 방법으로 해결하려고 했으며, 현재는 어떠한 아이디어를 사용하는지 ‘간략하게’ 살펴볼 예정입니다. 특히 Main으로 소개할 [ECCV 2016] ‘Colorful Image Colorization’ 논문에서는 Self-Supervised Representation Learning, Loss function 위주로 살펴보고 Colorization 을 활용하여 Computer Vision이나, Machine Learning 에서 어떻게 활용될 수 있는지 까지 살펴볼 예정입니다.
