Image Captioning

2017/07/21

발표자: 도진경

[발표 자료]

기존의 CNN만 사용하는 구조에서는 Classification에서 단순한 “단어”만을 뱉어내는 수준이었다면, time-series 기반의 RNN 구조를 결합하여 문장을 생성하게 함으로써 Image Captioning, Visual Questioning와 같은 분야에 응용할 수 있습니다.

CNN과 RNN이 결합된 구조의 Image Captioning에 대한 전반적인 소개와 함께, 특정 영역에 focus를 맞추도록 제안된 Attention Mechanism에 대하여 설명하겠습니다.
